# 使用node.js编写命令行工具

## 命令行工具
可以在命令行中直接输入一个命令，实现某个功能！

CLI: Command Line Interface  Vue-Cli


## 使用node.js编写命令行工具的流程
1. 先创建一个包
2. 在包中实现命令工具的功能
3. 在文件第一行写  #! node       告诉系统要使用node.js执行当前代码
4. 在package.json中添加 bin 属性   
    1. bin属性是一个对象，对象中的属性名就是最终要使用命令的名字
    2. 对象中的属性值，就是命令对应的要执行的文件的路径
5. 将当前包上传到npm服务器
6. 将这个包进行 全局安装
